# Playbook de Resposta a Brute Force

## Objetivo
Detectar e responder a tentativas de acesso não autorizado por força bruta.

## Passos
1. Identificar usuários com múltiplas falhas de login.
2. Bloquear temporariamente a conta suspeita.
3. Notificar o time de segurança.
4. Revisar logs para confirmar origem do ataque.
5. Implementar medidas adicionais (MFA, bloqueio de IP).
